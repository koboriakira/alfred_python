# alfred_python

Alfred用のスクリプトを保存するリポジトリ。

## 準備

このリポジトリを`~/git/`配下にcloneすること。

またライブラリのインストールが必須。

```shell
/usr/bin/python3 -m pip install -r requirements.txt
```
